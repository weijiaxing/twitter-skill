# 更新日志

- 初始版本发布
- 支持Twitter API v2发推功能
- 包含命令行工具接口
- 支持推文搜索和时间线获取

## 功能特性
- ✅ 发送单条推文
- ✅ 发布推文线程
- ✅ 搜索推文内容
- ✅ 获取用户时间线
- ✅ 点赞和转发操作

- ✅ 完整的错误处理
- ✅ 命令行接口支持

- Python 3.8+
- tweepy 4.0.0+
- Twitter API v2付费订阅
- Twitter API凭证（Consumer Key/Secret, Access Token/Secret）
